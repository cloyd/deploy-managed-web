# Managed App

ManagedApp is a powerful software solution designed to streamline and simplify the management of applications within your organization. It offers a wide range of features and functionalities that help you effectively manage your applications, improve efficiency, and enhance user experience.

## Run Locally

Clone the project

```bash
  git clone git@github.com:Managed-App/managed-web.git
```

Go to the project directory

```bash
  cd managed-web
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

Build for production

```bash
  yarn build
```

Locally preview the production build

```bash
  yarn preview
```
